# Тестовое задание в компанию Byndyusoft (Калькулятор)

## Установка зависимостей

```text
yarn
```

## Запуск калькулятора

```text
yarn dev
```

## Запуск тестов Jest + React Testing Library

```text
yarn test
```

## Запуск линтера Eslint

```text
yarn lint
```

## Запуск линтера Stylelint

```text
yarn stylelint
```
